Django + DRF

Commands

mkdir django_api
cd django_api
python -m venv .venv
. .venv/Scripts/activate
pip install django djangorestframework python-dotenv pydantic pydantic-settings
python -m django startproject core .
python manage.py startapp api
python manage.py runserver
