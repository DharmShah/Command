React + Tailwind CDN (Vite)

Commands

npm create vite@latest react_tailwind_cdn -- --template react
cd react_tailwind_cdn
npm install
# add Tailwind CDN script in index.html
npm run dev

Folder Structure

react_tailwind_cdn/
  index.html
  package.json
  src/
    main.jsx
    App.jsx
  .env.example
  README.md

index.html (Vite + Tailwind CDN)

<!doctype html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>React Tailwind CDN Vite</title>
  <script src="https://cdn.tailwindcss.com"></script>
  <script>
    tailwind.config = {
      theme: {
        extend: {
          colors: {
            brand: "#0f766e"
          }
        }
      }
    }
  </script>
</head>
<body>
  <div id="root"></div>
  <script type="module" src="/src/main.jsx"></script>
</body>
</html>
