# Commands Boilerplate Hub

Quickref-style command cards, now split across multiple HTML pages for faster loading.

## Open

- Main page: `index.html`
- Stack pages: `pages/`
- Boilerplate demo code: `boilerplates/`

## Includes

- Tailwind CDN on all docs pages
- Flask demo with database + template HTML + `.env`
- FastAPI demo with database + Pydantic settings + `.env`
- React demo with Tailwind CDN integration
- LangChain and LangGraph demos with API-key-ready `.env.example`
- Requirements files for each Python boilerplate
